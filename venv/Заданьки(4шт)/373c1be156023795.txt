Задание Иван 	

Тестовая часть
1(V). Создать программу, которая будет находить медиану списка из 20 чисел. Числа должны быть сгенерированы случайным образом. Если количество элементов в списке четное, медианой считается среднее значение двух центральных элементов, если нечетное - центральный элемент.

2(V). Написать функцию, которая принимает на вход список строк и возвращает новый список, состоящий из строк, содержащих только уникальные символы. Например, если на вход подается список ["hello", "world", "python"], то функция должна вернуть список ["heo", "wrd", "pytn"].

3(V). Напишите программу на Python, которая принимает на вход строку и выводит все ее подстроки, отсортированные в лексикографическом порядке. Например, для строки "abc" результатом работы программы должны быть строки "a", "ab", "abc", "b", "bc", "c".

4(V). Напишите программу для поиска наибольшего общего делителя (НОД) двух чисел с помощью алгоритма Евклида. Программа должна принимать два целых числа и возвращать их НОД.

Проектная часть

Создайте чат-бота, который может обрабатывать запросы на заказ продуктов из ресторана.
Бот должен иметь функциональность для выбора продуктов из меню, добавления и удаления продуктов в корзине, указания адреса доставки и оплаты заказа.
Также бот должен уметь обрабатывать запросы на изменение заказа и отмену заказа.

Бот перед работой с клиентом должен собрать информацию о нем

Бот должен умень ставить товар на стоп
Бот должен уведомить пользователя о времени доставки (время макс времязатратного блюда + 30 минут) (см прил 2)


Бот работать с группой администраторов (создать группу и отлавливать из нее сообщения (группа для обоих ботов в тг))
Группа администраторов - является не только группой пользователей но и чатом, в который должны присылаться уведомления о вещах требующих их участия
Бот должен иметь панель управления для администраторов
Бот должен иметь возможность добавления/изменения/удаления администраторов
Реализовать 2 уровня прав администраторов

Бот должен работать на 2х платформах (vk и tg)| см приложение 1
Бот должен иметь базу данных на sqlite

Приложение должно иметь как оконный интерфейс для работы с товарами, так и клавиатурный 
Все админ панели только для бота в тг, без вк

Бот должен собирать историю изменений его настроек
Отдельный json файл с ключами и любыми другими фиксированными переменными, которые в будущем могут потребовать изменения

Бот должен собирать статистику заказов, давать возможность пользователю оставить коментарий, как к блюду так и к заказам
Показывать коментарии к блюду в случае одобрения администратором
Бот должен иметь систему оценки блюда, по умолчанию 4* из 5, стойкость 10


Бот должен уметь востановить свою работу с исходными данными в случае сбоя
Проект должен иметь осмысленную файловую структуру
Предоставить проект в виде гита или укомплектованного архива и быть готовым к его презентации
